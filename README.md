# KaamKarDo Ecosystem

Welcome to **KaamKarDo**, the #1 AI platform for professionals to get hired faster and look professional.

## Platform URL
[KaamKarDo.com](https://kaamkardo.com)

## The Ecosystem
Our platform offers a scalable ecosystem of micro-SaaS tools designed to accelerate your career growth:

- **[PhotoReady](https://photoready.kaamkardo.com/)**: India's AI Style Marketplace and Image Generator. Generate studio headshots, wedding portraits, dating profile pictures, and reels seamlessly.
- **[KaamKarDo Resume](https://resume.kaamkardo.com/)**: Paste any Job Description and get a resume optimized to beat ATS in seconds. 30-Day Job Hunt OS included.
- **SkillHub**: Personalized skill roadmaps to accelerate your career.
- **Social**: Generate viral LinkedIn posts that build your personal brand.
- **SEO Writer**: Write SEO-optimized articles that drive organic traffic.

## Tech Stack
- Framework: Next.js 16 (App Router)
- Language: TypeScript
- Styling: Tailwind CSS v4 & Framer Motion
- UI Components: shadcn/ui
- Deployment: Vercel

## Running Locally

First, install the dependencies and then run the development server:

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
